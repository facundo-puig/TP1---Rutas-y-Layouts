## Dev Store - Tienda de Componentes de PC

Este proyecto está enfocado en una página web para una tienda de venta de componentes de PC con un diseño "gaming". Cuenta con la integración de la API de [text](https://www.hardgamers.com.ar/) a tráves de un scraper para obtener los productos disponibles.

## Instrucciones para correr el proyecto

Clonar repositorio
```bash
git clone <repo>
```

Entrar al proyecto
```bash
cd tp1-grupal
```

Instalar dependencias
```bash
npm install
```

Ejecutar
```bash
npm run dev
```
